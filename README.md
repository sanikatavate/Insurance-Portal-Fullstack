# Insurance Portal - Full Stack Application

A modern full-stack insurance management platform that helps users browse, purchase, and manage insurance policies online.

## Project Structure

```text
Insurance-Portal--Fullstack
├── Backend/
├── Frontend/
└── README.md
```

## Features

* User Authentication & Authorization
* Insurance Policy Management
* Secure Payment Integration
* User Dashboard
* Claims Management
* Document Management
* Notifications System
* AI Support Assistant
* Admin Dashboard
* Agent Dashboard

## Frontend

The Frontend is built using modern web technologies and provides a responsive user interface for customers, agents, and administrators.

### Technologies

* React.js
* Vite
* JavaScript
* CSS

## Backend

The Backend handles business logic, authentication, APIs, database operations, and policy management.

### Technologies

* Node.js
* Express.js
* MongoDB

## Installation

### Frontend

```bash
cd Frontend
npm install
npm run dev
```

### Backend

```bash
cd Backend
npm install
npm start
```

## Environment Variables

Create a `.env` file inside the Backend directory and configure the required environment variables.
